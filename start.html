01

Allume le capteur

Sur batterie : il est déjà actif. Sur secteur : branche le câble USB. Un bip court confirme le démarrage.

02

Trouve le réseau du capteur

Sur ton téléphone : Réglages → Wi-Fi. Cherche un réseau qui commence par

SoDetect-XXXXXX

Connecte-toi — pas de mot de passe requis à cette étape.

03

Choisis ton Wi-Fi

Une page s'ouvre automatiquement. Sélectionne le Wi-Fi du bâtiment, entre son mot de passe, valide. Le capteur redémarre seul.
